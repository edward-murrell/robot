# Robot

# Tests
All tests can be run by calling:
```
make tests
# OR
python3.6 -m unittest tests
```


## Unit
To run unit tests, use the following shell commands:
```
make unit.tests
# OR
python3.6 -m unittest tests.unit
```

## Limitations
- Board height & width can be zero, or negative.
